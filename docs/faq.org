* FAQ
** Where's the name coming from
Evenor stands for "EVENt editOR". In Greek, Evenor is the ancestor of the people of Atlantis. In Tolkien's Elvish, "evenor" can be loosely translated as "common ground".

people also call it `evernor` and `everon`.

** how to allow your git instance
- add https://evenor.norcivilianlabs.org to CORS
- if you use Anubis, add it to exceptions
- create an access token in your git instance
- if you already have a dataset in evenor, add origin_url and origin_token. evenor will push to bare repo.
- or you can import an example to your git
- once you have a git repo, you can access it from any evenor with https://evenor.norcivilianlabs.org/#?~=url&-=token
